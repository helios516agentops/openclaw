# Project Structure

## Top-Level Directories

### Core Source

- **`src/` (71 items)** — Main TypeScript source code
  - Agent Runner implementation
  - Model Resolver with failover logic
  - System Prompt Builder
  - Session History Loader (JSONL format)
  - Core messaging channel adapters

### Extensions & Skills

- **`skills/` (53 items)** — Extensible skill system
  - Plugin architecture for custom behaviors
  - ClawHub marketplace integration
  - Built-in skills (e.g., gog for Google Workspace)

- **`extensions/` (38 items)** — Extension modules
  - Additional functionality beyond core skills
  - Third-party integrations

### Applications

- **`apps/` (6 items)** — Platform-specific applications
  - Web, mobile, desktop variants

- **`Swabble/` (15 items)** — Swift macOS/iOS native app
  - Native Apple platform integration
  - AppKit/SwiftUI implementations

- **`packages/` (4 items)** — Monorepo packages
  - Shared libraries across workspace
  - Internal utilities

### User Interface

- **`ui/` (10 items)** — Control panel UI
  - Built with Lit 3.3.2 (Web Components)
  - Dashboard for skill management
  - Configuration interface

### Testing & Documentation

- **`test/` (15 items)** — Test suite
  - Vitest 4.0.18 test framework
  - Unit + integration tests

- **`docs/` (44 items)** — Documentation
  - API references
  - Skill development guides
  - Deployment instructions

### Build & Deployment

- **`scripts/` (79 items)** — Helper scripts
  - Build automation
  - Deployment utilities
  - Development workflows

### Configuration Files

- **`AGENTS.md`** — AI agent definitions
- **`CLAUDE.md`** — Symlink to AGENTS.md (agent instructions)
- **`Dockerfile`** — Container image definition
- **`docker-compose.yml`** — Multi-container orchestration
- **`Dockerfile.sandbox`** — Sandboxed execution environment
- **`package.json`** — Workspace root configuration
- **`tsconfig.json`** — TypeScript compiler settings
- **`.agents/`** — AI agent definition files

## Runtime Configuration

- **`~/.openclaw/openclaw.json`** — User configuration
  - LLM provider API keys (via OAuth)
  - Channel credentials
  - Skill settings
  - Session preferences

## Key Entry Points

- **`openclaw.mjs`** — Main executable (generated by build)
- **`src/index.ts`** — TypeScript entry point
- **`ui/index.html`** — Control panel entry
